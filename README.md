# Walmart Inventory Optimization & Sales Forecasting

## Business Problem
Walmart faces inventory inefficiencies due to rising holding costs and uneven store performance.

## Dashboard Features
- KPI Overview
- Sales vs Holding Cost Trend
- Store-Level Analysis
- Safety Stock Analysis
- CPI Impact

## Key Insights
- Holding costs rose faster than sales in 2012.
- Some stores incur disproportionately high holding costs.
- Safety stock misalignment identified.

- Power BI
- DAX
